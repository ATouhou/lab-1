Computer

ReactOS An open source version of Microsoft's infamous Windows operating system. They are making good progress and have a live image that can be downloaded.
Asterisk PBX An open source VoIP PBX. I have used this in conjunction with the Cisco 7960 phones, Cisco ATA-186s, and various VoIP providers to run a fully functional in house PBX. This project is very mature and is used by businesses for communicating with customers.
Flat Feet Pete Pete is a friend of mine and a brilliant programmer. Check out his web site for lots of code, and crazy projects he has come up with.
