AT&T Wireless GPRS:

The AT&T Wireless GPRS system is super easy to configure on a phone like the Nokia 7210. However the ATTWS customer support people are not very helpful. I guess they think everyone should buy a phone from AT&T. Thanks but I couldn't wait for a decent GSM phone.
  • Setting's name can be anything but I named it AWS WAP
  • Homepage http://home
  • Session mode is Permanent
  • Connection security is On
  • Data bearer is GPRS
  • GPRS access point is left blank
  • IP Address 10.250.250.250 (AT&T said it was 01.250.250.250)
  • Authentication type is Normal
  • Login type is Automatic
  • Username and Password are blank

    Java IP Subnet Calculator:

    I'm currently working on an IP subnet calculator in Java. My plan is to have it running on all platforms and my Nokia 7210. The Nokia 7210 is Java MIDP capable so I am excited to be able to use my phone for subnetting. It will be open-source.
